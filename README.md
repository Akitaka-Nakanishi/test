# 二重交差検証(Double Cross Validation)

0. サンプルデータを作成
1. k最近傍法 (k-Nearest Neighbor, k-NN)で分類
2. モデルの適用領域 (Applicability Domain, AD)をk-NNで計算
3. 二重交差検証 (Double Cross Validation)でその予測性能を推定。二重交差検証は複数回実行し、その平均と分散を表示。
